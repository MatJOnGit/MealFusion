MealFusion API Documentation

Welcome to the official documentation for the MealFusion API. This API allows you to access a variety of ingredients and recipes for your meal planning needs.

Authentication

To access the MealFusion API, you need to include your authentication key in the headers of your requests. Your key can only be obtained by subscribing to a nutrition program on the Dodie Coaching website right now.

Header parameter format:

  • Content-type

                        
                        application/json

  • Authorization

                        
                        Bearer {APIkey}

Endpoints

Here are the available endpoints for the MealFusion API:

Get Ingredient by ID

Returns the details for the ingredient with the specified ID.

                
                    GET http://api.mealfusion.fr/v1/recipes?id={ingredient_id}

Parameters:

  • ingredient_id (required) - The unique ID of the ingredient to retrieve.
  • authentification (required) - Your API key.
                
                    GET http://api.mealfusion.fr/v1/recipes?id=123

Get Ingredients by Name

Returns a collection of ingredients whose name contains the specified string, along with their nutritional data.

Note : every word must be separated by an underscore

                
                    GET http://api.mealfusion.fr/v1/recipes?name={ingredient_name}

Parameters:

  • ingredient_name (required) - The string to search for in ingredient names.
  • authentification (required) - Your API key.
                
                    GET http://api.mealfusion.fr/v1/recipes?name=salade_ni√ßoise

Get All Ingredients

Returns a collection of all ingredients, along with their nutritional data.

                
                    GET http://api.mealfusion.fr/v1/ingredients

Parameters:

  • authentification (required) - Your API key.
  • Content-Type (required) - "application/json"

Get Recipe by ID

Returns the ingredients and measurements for the recipe with the specified ID.

                
                    GET http://api.mealfusion.fr/v1/recipes/{recipe_id}

Parameters:

  • recipe_id (required) - The unique ID of the recipe to retrieve.
                
                    GET http://api.mealfusion.fr/v1/recipes?id=456

Get Recipes by Name

Returns a collection of recipes whose name contains the specified string.

Note : every word must be separated by an underscore

                
                    GET http://api.mealfusion.fr/v1/recipes?name={recipe_name}

Parameters:

  • recipe_name (required) - The string to search for in recipe names.
                
                    GET http://api.mealfusion.fr/v1/recipes?name=couscous

Get All Recipes

Returns a collection of all recipes.

                
                    GET http://api.mealfusion.fr/v1/recipes

Parameters:

  • recipe_name (required) - The string to search for in recipe names.
